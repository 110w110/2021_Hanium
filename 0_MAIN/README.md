### Aduino_Streaming.ino
아두이노 보드인 ESP32를 전반적으로 컨트롤하는 ino 파일입니다. ESP-32에 현재 탑재되어 있으며 변경된 프로그램을 업로드하여 작동할 수 있습니다.  
보드가 부팅되면 AutoConneted AP가 활성화 되며, 타 기기를 활용하여 ESP-32를 특정 AP에 접속시킬 수 있습니다.
